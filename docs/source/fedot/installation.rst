Installation from GitHub
========================
  | git clone https://github.com/nccr-itmo/FEDOT.git
  | cd FEDOT
  | pip install .
  | pytest -s test/unit

or full installation

  | git clone https://github.com/nccr-itmo/FEDOT.git
  | cd FEDOT
  | pip install .[extra]
  | pytest -s test/unit

Installation from PyPI
======================
  | pip install fedot  # common
  | pip install fedot[extra]  # full
